Git Basics

playing with git

Install

Download & Install git in your machine from http://git-scm.com/

Set Config

We might set some basic git config for my machine:-

$ git config --global user.name "Asad Rahman"
$ git config --global user.email "asad.dk.bd@gmail.com"
$ git config --color.ui true

Help

To show list of all git commands use help:-

$ git help

To get details of a particular command use git help format

$ git help init

Initiate git for a project

Move to your destination directory and use git init

$ cd my_project_dir
$ git init

Check git status. Git status gives you list of untracked files, list of modified files and current commit head

$ git status

Playing with some basic commands

Add a file "index.php" to staging

$ git add index.php

Check git status, will show index.php as tracked new file in staging.

Make your first commit:-

$ git commit -m "Here put some message"

Check git status again

Check git log for commit history

$ git log

List all brunches and check the current brunch 

$ git brunch

Create a new brunch

$ git brunch my-first-brunch-name

Check the brunch list again

Lets move to the new created brunch, use git checkout command:-

$ git checkout my-first-brunch-name

Add some more files here and do some commits in this brunch.

Play with git add command

Add all files

$ git add --all

Add all .txt files in current directory

$ git add *.txt

Add all .txt files in docs directory

$ git add docs/*.txt

Add all files in docs directory

$ git add docs/

Add all .txt files in all directory & sub directories

$ git add "*.txt"
